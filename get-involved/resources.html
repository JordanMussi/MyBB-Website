MyBB Documentation

Contribute to the MyBB Documentation using GitHub

MyBB uses a GitHub repository to manage the official MyBB Documentation, allowing MyBB users and staff to easily collaborate and contribute to the documentation. To make contributions, you just need to fork the mybb.github.io repository, add or update pages, push the changes to your forked repository, then send a pull request for review. Once the pull request is approved, your changes will be merged into the official MyBB Documentation.

More information on making contributions to the documentation is available on the MyBB Documentation repository on GitHub.
